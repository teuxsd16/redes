Atividade 01

1- Ponto a ponto é a conexão de dois computadores usando um meio de comunicação, é a mais simples. Vantagens é a velocidade de dados e as desvantagens é que a falha de um PC afeta tudo.   Na de barramento todos os computadores são ligados em um mesmo barramento físico de dados. Apenas um computador envia dados e os outros esperam. Se várias máquinas enviares dados simultaneamente ocorre colisões. As vantagens são a maior expansão e economia, já as desvantagens são o trafego pesado e uma falha no cabo para o sistema.
A anel é a conexão de computadores em série, fechando um ciclo. Os dados são em uma única direção. As vantagens são o equilíbrio de acesso e a alta performance. A desvantagem principal é que se um computador parar, o sistema todo para.
A estrela é a mais usada, em que são vários computadores ligados ao um ponto central, esse último se encarrega de retransmitir os dados.  A vantagem é o fácil gerenciamento da rede e que uma falha em um PC não afeta o sistema. Mas como desvantagem é que um erro no conector central para a rede.

2- A internet é uma rede de computadores mundiais que permite o compartilhamento de informações em qualquer parte do mundo. Semelhante a esta última, a intranet é uma rede privada baseada em protocolos, mudando apenas da internet a restrição de acesso. Já Extranet é semelhante também a internet diferenciando no acesso aberto sem bloqueios.

3- A primeira camada é a física que tem a função de receber e enviar bits através de um meio transmissor. A segunda é a de enlace de dados, funcionando para garantir a integridade dos dados e detecção de erros. Depois a de rede, suas principais funções são o roteamento dos pacotes entre fonte e destino. As principais funções da camada de transporte são a estabelecer conexões para cada requisição vinda da camada superior. A de sessão é organizar diálogos entre dois processos aplicações. Cabe a camada de apresentação a tradução de dados e formatação do código, também exibi dados para a camada de aplicação. Por fim, a camada de aplicação é o local onde o usuário se comunica com o computador, estabelecendo uma interface entre o software e as camadas inferiores.

4- A cama de aplicação estabelece a conexão entre o protocolo de transporte e as aplicações. Já a camada de acesso a redes estabelece o modelo em que os dados devem ser transmitidos. A de internet vai fornecer os pacotes. Quem tem a função de transforma os dados transmitidos da camada de aplicação em pacotes é a camada de transporte, que além disso repassa esses pacotes para a camada de internet. 

5-  O hub é um dispositivo simples que interliga vários computares. Ele recebi os dados de uma máquina e repassa as outras. Quando uma máquina envia sinais, as outras tem que esperar para enviar também, assim o hub fica em estado de espera. Ele opera na camada física do modelo OSI. Já o switch muda apenas que quando uma máquina envia dados o mesmo é repassado a só uma máquina de destino. Dessa forma, o trafego de dados está sempre livre acelerando o desempenho da rede. Seu funcionamento é na camada 2, a de enlace de dados. O roteador é mais próximo do switch, conectando vários computadores e escolhendo o melhor caminho para o trafego de dados, e opera na camada de rede.
